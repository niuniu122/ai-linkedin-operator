# Tools

This directory contains lightweight wrapper scripts and OpenCLI overrides.

The full OpenCLI and browser-harness source trees are intentionally not committed here. Run:

```powershell
.\scripts\install.ps1
```

That creates a local ignored `tool/` directory with:

- `tool/OpenCLI`
- `tool/browser-harness`
- `tool/opencli.ps1`
- `tool/browser-stack.ps1`
- `tool/browser-harness.ps1`
- `tool/browser-harness-isolated.ps1`
